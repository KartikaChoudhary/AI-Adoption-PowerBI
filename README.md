# AI Adoption in Each Sector — Power BI Dashboard

## Overview
Power BI dashboard analyzing AI adoption rates, investment trends,
and use cases across 10 industry sectors.

## Tools Used
- Microsoft Power BI Desktop
- Power Query & DAX
- Data Sources: McKinsey, Stanford AI Index, Statista

## Files
- `AI_Adoption_Analysis.pbix` — Power BI dashboard
- `AI_Adoption_Report.pdf` — Project report/presentation

## Key Insights
- Technology leads AI adoption at 78%
- Agriculture fastest growing at +30% YoY
- Global AI market valued at $305B in 2024
